## Space-Vim
https://spacevim.org/

Fix the question mark showing by installing a compatible font
```
brew cask install font-hack-nerd-font
brew tap homebrew/cask-fonts

In iTerm2 go Preferences -> Profiles -> (Profile) -> Text -> Font -> Hack Nerd Font Mono
```
